Hermes Project : autonomous vehicule teleoperation

The Hermes Project is an application that allows a vehicule to be teleoperated.

From the intereface the teleoperator can connect itself to the vehicule (thanks to the vehciule's IP address), he can then create a mission (end point of the mission, maximum speed ... ), and then he sends the mission to the car. He can start the mission whenever he wants, stop it or abort it too. When "Start" is pressed on the interface, the vehicule begins the mission. Inside the vehicule, various controller have been implemented to be sure that the vehicules follows the right trajectory with the right speed.

When the vehicules encounters an obstacles, the obstacle is detected thanks to a LiDar, if the obstacle is on the vehicule path, it stops, and ask for help to the teleoperator. The teleoperator can now define a new trajectory in order to avoid a collision The new path definition implements Bezier algorithm, we've also simulated the vehicule's width so that the teleoperator knows if the new path really avoid a collision
